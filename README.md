## Cara Instalasi:
1. Download file `64-bit.INDOKU.zip / 32-bit.INDOKU.zip` di bawah ini.
2. Tempatkan di folder mana saja (disarankan di Desktop).
3. Jalankan aplikasi, arahkan ke folder GTA San Andreas Anda, dan tekan **Play**!
